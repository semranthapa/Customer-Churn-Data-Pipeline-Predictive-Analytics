# Customer-Churn-Data-Pipeline-Predictive-Analytics

## Overview 
- Data cleaning and transformation workflows
- Logistic regression model training and evaluation
- Precision@K evaluation for business targeting
- Probability-based churn risk scoring
- Power BI dashboard for revenue-at-risk analysis

## Model Performance

- **ROC-AUC:** 0.83  
- **Precision@200:** 69%

This model is designed to identify high-risk churn customers for targeted retention strategies.
